Prime Hub V4.0.2

Changes:
- Order history now shows product name, quantity, status, total, date and time.
- Delivery messages now show order ID, product, quantity, date and time.
- Each delivered stock item has a cleaner separate block.
- The store title is normalized to “Prime Hub Store”.
- Category buttons show total live stock for that category.
- All Products shows total live stock across active products.
- Product-list status uses 🟢 and 🔴 round indicators instead of check/cross icons.

Telegram does not support custom button background colors, so the update uses round
green/red indicators while preserving the existing Telegram button style.
