Prime Hub V4.0.5 Category Management

New:
- /moveproduct PRODUCT_ID: move a product to an existing category using buttons.
- /deletecategory: safely remove empty/accidental categories.
- /addproduct now shows existing category buttons instead of requiring free-text category entry.
- Slash commands are rejected as category names in the fallback flow.

For the current LinkedIn product:
1. Run /listproducts and note the LinkedIn Premium product ID.
2. Run /moveproduct PRODUCT_ID.
3. Tap Learning & Courses.
4. Run /deletecategory and select /listproducts if it still appears empty.
