Prime Hub V3.7 — Support, AI assistant, restock alerts, and community updates

New customer commands:
- /help — interactive support menu
- /tickets — customer ticket history
- /assistant — optional AI store assistant
- /community — Prime Hub updates link

New admin commands:
- /replyticket TICKET_ID MESSAGE
- /announce MESSAGE

New Railway variables:
- UPDATE_CHAT_IDS=-1001234567890,@PrimeHubUpdates
- COMMUNITY_LINK=https://t.me/PrimeHubUpdates
- OPENAI_API_KEY=...   (optional)
- OPENAI_MODEL=gpt-5-mini

For channel posting, add the bot as an administrator with permission to post messages.
For groups, add the bot as a member and allow it to send messages.
